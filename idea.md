# GreenCart - Project Idea

## Project Scope
GreenCart is a modern full-stack e-commerce platform specializing in organic and fresh produce. It provides a seamless shopping experience for customers to browse, cart, and purchase groceries, while also offering a dedicated dashboard for sellers to manage products and orders.

## Key Features
- **Categorized Shopping**: Browse items by categories like Vegetables, Fruits, Dairy, and Bakery.
- **Dynamic Promotions**: Flash Sales and Best Seller sections to highlight top products.
- **User Authentication**: Secure login and registration for both customers and sellers.
- **Cart Management**: Real-time cart updates with persistent storage.
- **Order Tracking**: Detailed order history and status tracking.
- **Address Management**: Save multiple shipping addresses for faster checkout.
- **Seller Dashboard**: Interface for sellers to add products, track stock, and manage customer orders.
- **Responsive Design**: Premium UI optimized for both desktop and mobile devices.
